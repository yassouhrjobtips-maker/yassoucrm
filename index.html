⚙️ Sauvegarde des données (Supabase)

Sans Supabase, tes données restent uniquement dans ce navigateur (localStorage). Connecte un projet Supabase gratuit pour conserver tes clients, accompagnements et notes en ligne, accessibles depuis n'importe quel appareil.

Ces deux valeurs se trouvent dans Supabase : Project Settings → API. Crée la table de stockage avec le script SQL fourni (fichier supabase-setup.sql) avant de te connecter.
